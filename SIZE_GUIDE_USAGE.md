# Size Guide - Quick Usage Guide

## Customer View

### How to Access the Size Guide

1. **Navigate to Product Page**
   - Go to any product detail page (e.g., `product.html?id=xxx`)

2. **Find the Size Guide Button**
   - Look for the "Size Guide" link next to "SELECT SIZE"
   - The button has a ruler icon (📏) and blue text

3. **Open the Modal**
   - Click the "Size Guide" button
   - A large modal window will appear with all sizing information

## What's Inside the Size Guide

### Section 1: How to Measure (Top Section)
Visual cards showing how to measure:
- **Chest**: Measure around fullest part
- **Waist**: Measure around natural waistline
- **Hips**: Measure around fullest part
- **Shoulder**: Measure from edge to edge
- **Sleeve Length**: Measure from shoulder to wrist
- **Length**: Measure from shoulder to hem

Each card has:
- A circular icon with the measurement type
- Clear instructions
- Hover effect for better visibility

### Section 2: Measurement Tips (Yellow Box)
Important tips including:
- Use a soft measuring tape
- Wear fitted clothing
- Keep tape parallel to floor
- Don't pull too tight
- Ask for help if needed

### Section 3: Men's Size Chart
Complete table with sizes XS to XXL showing:
- Chest measurements (inches)
- Waist measurements (inches)
- Hips measurements (inches)
- Shoulder measurements (inches)
- Sleeve measurements (inches)

### Section 4: Women's Size Chart
Complete table with sizes XS to XXL showing:
- Bust measurements (inches)
- Waist measurements (inches)
- Hips measurements (inches)
- Shoulder measurements (inches)
- Sleeve measurements (inches)

### Section 5: Fit Guide
Three fit types explained:
- **Slim Fit**: Modern, streamlined look
- **Regular Fit**: Classic, comfortable fit
- **Relaxed Fit**: Loose and comfortable

### Section 6: International Size Conversion
Conversion table for:
- US sizes (XS-XXL)
- UK sizes (6-16)
- EU sizes (34-44)
- Japan sizes (S-4L)

## Common Use Cases

### Use Case 1: First-Time Buyer
1. Open size guide
2. Read "How to Measure" section
3. Measure yourself following instructions
4. Compare with size chart
5. Select appropriate size

### Use Case 2: International Customer
1. Open size guide
2. Scroll to "International Size Conversion"
3. Find your country's size system
4. Convert to US size
5. Verify with measurement chart

### Use Case 3: Between Sizes
1. Open size guide
2. Check measurements for both sizes
3. Read "Fit Guide" section
4. Choose based on preferred fit:
   - Smaller size for slim fit
   - Larger size for relaxed fit

### Use Case 4: Gift Shopping
1. Open size guide
2. Print or screenshot the measurement instructions
3. Measure recipient's similar clothing
4. Compare with size chart
5. Select size

## Tips for Best Results

1. **Measure Twice**: Always double-check measurements
2. **Use Existing Clothes**: Compare with well-fitting items
3. **Consider Fabric**: Stretchy fabrics may fit differently
4. **Read Product Description**: Check for fit notes
5. **When in Doubt**: Size up for comfort

## Troubleshooting

### Modal Won't Open
- Check if JavaScript is enabled
- Verify Bootstrap is loaded
- Clear browser cache
- Try different browser

### Tables Not Displaying
- Enable JavaScript
- Check internet connection
- Refresh the page
- Try desktop view on mobile

### Can't See All Content
- Scroll within the modal
- Use full-screen mode
- Rotate device to landscape
- Zoom out if needed

## Quick Reference

### Size Selection Flow
```
1. Click "Size Guide" button
   ↓
2. Read measurement instructions
   ↓
3. Measure yourself
   ↓
4. Find your measurements in chart
   ↓
5. Note your size
   ↓
6. Close modal
   ↓
7. Select size on product page
   ↓
8. Add to cart
```

### Measurement Order
```
Chest/Bust → Waist → Hips → Shoulder → Sleeve → Length
```
